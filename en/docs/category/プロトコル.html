プロトコル

ファイルサーバにアクセス可能なプロトコル一覧

📄️ API

Fastifyを用いてAPIサーバーを実装し、WebUIと合わせて使用することができる。

📄️ FTP

ベースとなるパッケージとしてftp-srvを用いる。

📄️ SFTP

ベースとなるパッケージとしてssh2を用いる。

📄️ TFTP

ベースとなるパッケージとしてnode-tftpを用いる。

📄️ WebDAV

ベースとなるパッケージとして`webdav-server`を用いる。
